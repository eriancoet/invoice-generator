# Invoice Generator

🚀 **Live Demo:**
👉 [https://invoice-generator-jade-eight.vercel.app/](https://invoice-generator-jade-eight.vercel.app/)

Invoice Generator is a full-stack web application built with **Next.js (JavaScript)** and **Supabase**, designed to help users create, manage, and export professional invoices quickly and efficiently.

The app provides secure authentication, real-time data storage, and a clean, responsive interface optimized for business use.

---

## ✨ Features

* 🔐 Secure authentication with Supabase
* 🧾 Create and manage invoices
* ➕ Add and edit line items (quantity, rate, totals auto-calculated)
* 📄 Professional invoice layout
* 🖨 Print-ready invoice view
* 💾 Persistent invoice storage
* 📱 Fully responsive design

---

## 🛠 Tech Stack

* **Framework:** Next.js
* **Language:** JavaScript
* **Backend & Database:** Supabase
* **Authentication:** Supabase Auth
* **Deployment:** Vercel

---

## ⚙️ Getting Started

### 1️⃣ Install Dependencies

```bash
npm install
```

### 2️⃣ Run Development Server

```bash
npm run dev
```

Open:

```
http://localhost:3000
```

---

## 🔑 Environment Variables

Create a `.env.local` file in the root directory:

```
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
```

Restart the development server after adding environment variables.

---

## 📦 Build for Production

```bash
npm run build
npm start
```
